# Convert audio into text with speakers

to convert audio to text.

- get the audios from one path named "input"

- convert it into text file and save to path named "output" as the filename the same with the audio name but with .txt extention.
- move the converted audio files into the path named "done" and remove origin one if move successfully.
- move the failed audio files into the path named "failed" and remove origin one if move successfully.
- merge same speaker and times' text into one line.
